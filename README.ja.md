# domtest

このリポジトリには、DOM操作の基本的な機能を実装したサンプルコードが含まれています。ブラウザのDOM API を模擬したシンプルなライブラリを提供し、Deno上でのDOM操作のテストを可能にします。

## デモ
このライブラリを使ったサンプルコードは、`domtest.test.js` に記載されています。Denoを使ってテストを実行することで、ライブラリの動作を確認することができます。

## 機能
- DOM要素の作成
- 要素へのテキストコンテンツの設定
- 要素へのリンクの設定
- 要素の追加

## 必要環境
- Deno 1.x

## 使い方
1. Denoをインストールします。
2. リポジトリをクローンします。
3. `deno test domtest.test.js` コマンドを実行してテストを実行します。
