# domtest

このリポジトリには、DOM操作の基本的な機能を実装したサンプルコードが含まれています。ブラウザのDOM APIを模擬したシンプルなライブラリを提供し、Deno上でのDOM操作のテストを可能にします。

## 機能
- DOM要素の作成
- 要素へのテキストコンテンツの設定
- 要素へのリンクの設定
- 要素の追加

## 必要環境
- Deno 1.x

## 使い方
1. Denoをインストールします。
2. リポジトリをクローンします。
3. `deno test domtest.test.js` コマンドを実行してテストを実行します。
